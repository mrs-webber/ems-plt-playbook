### Agenda

| Time | Focus |
| --- | --- |
| 0–5 min | Review the draft CFA. |
| 5–20 min | Check each question/task against the learning target. |
| 20–30 min | Agree on scoring expectations. |
| 30–38 min | Decide how data will be recorded by student and by target. |
| 38–45 min | Set the assessment window and data review date. |

### Why This Meeting Matters

This prevents teams from giving an assessment that is common in name only. The team needs to know exactly what each item measures and how results will be used.
